# 指示書：牛舎画面の実装（フェーズ1）

## これは何の作業か

プロトタイプ（ushiya_v3.html）を土台に、ゲームの本実装として牛舎画面を作る。
フェーズ1のゴールは「牛舎に入って、牛をタッチして詳細シートが開く」までを動作させること。

---

## 前提

- プロトタイプ（ushiya_v3.html）を参照・流用してよい
- 背景画像（barn_bg）はプロトタイプのものをそのまま使う
- 牛のブチ模様生成ロジック（シード値→模様）はプロトタイプのものをそのまま使う
- 主人公の歩行アニメもプロトタイプのものをそのまま使う
- セーブデータ（yamayama_loop_v1）から牛の情報を読み込む
- **文字列はすべてt()経由で取得すること（ja.jsonに追記）**
- **内部名称は英語で統一**

---

## 牛房のレイアウト

背景画像は左右2列×4行＝8部屋の牛房。上から：

| 行 | 左 | 右 |
|---|---|---|
| 1行目 | 母牛①（s0） | 母牛②（s1） |
| 2行目 | 母牛③（s2） | 空き予備（s3） |
| 3行目 | 子牛①（s4） | 子牛②（s5） |
| 4行目 | 子牛③（s6） | 子牛④（s7） |

現時点では母牛スロットは3つ（s0〜s2）、子牛スロットは3つ（s4〜s6）まで使用。
s3・s7は将来用として空きのまま表示。

---

## 各牛房に表示する要素

### 牛がいる牛房

- 牛のスプライト（ブチ模様生成済み、または通常スプライト）
- 牛の名前ラベル（小さく）
- 💩アイコンエリア（右下に最大4つ並ぶスペースを確保。初期値は非表示）
- 😷アイコン（左上に表示スペースを確保。初期値は非表示）

### 空き牛房

- 干し草の表現（プロトタイプ通り）
- 「空き」ラベル

---

## 💩と😷のオンオフ管理

**最初から実装すること。** ただし現時点ではすべて非表示（false）でよい。
将来のフェーズで表示ロジックを追加しやすくするための枠として用意する。

セーブデータの各牛に以下のフィールドを追加：

```js
{
  // 既存フィールド
  id: "cow_001",
  name: "ふうか",
  // ...

  // 今回追加（初期値）
  poop_count: 0,          // 💩の数（0〜4）
  disease_alert: false,   // 😷アイコン表示フラグ
}
```

描画時：
- `poop_count > 0` のときだけ💩を `poop_count` 個描画
- `disease_alert === true` のときだけ😷を描画
- 現時点では両方 false / 0 なので何も表示されない

## タップ（タッチ）処理

### 牛房タップ
- 牛がいる牛房をタップ → 主人公が歩いて近づく → 詳細シートを開く
- 空き牛房をタップ → 主人公が歩いて近づく → 「空き牛房です。草を敷いて待っています。」シートを開く

### 詳細シートの内容（牛がいる場合）

```
[牛のキャンバス画像]  ふうか
                    24ヶ月
                    体調 ████████ 8
                    品質 可
                    👦 牧童の目

─────────────────────
毛艶コメント（品質ポイント%換算）
```

毛艶コメントのロジック（既存のまま流用）：
- 0〜25%：「最近少し元気がないかも…」
- 26〜50%：「普段通りのふうかだ。」
- 51〜75%：「毛艶がよくなってきた気がする。」
- 76〜99%：「なんだかいい感じ…そろそろかも？」

## ホーム画面との接続

- ホーム画面の「牛舎」ボタン → `barn.html`（または同等のページ）へ遷移
- 牛舎画面の「戻る」ボタン → `home.html`へ遷移
- セーブデータの読み書きは `yamayama_loop_v1` キーで統一

---

## ファイル構成

```
barn.html          ← 今回新規作成
assets/js/barn.js  ← 牛舎ロジック（common.jsをimport）
assets/js/common.js ← 既存（変更最小限）
assets/i18n/ja.json ← 今回追加するキーを末尾に追記
```

## 触らないこと

- 💩の増加ロジック（フェーズ2で実装）
- 😷の発動ロジック（フェーズ2で実装）
- 種付け・発情・出産のロジック（別フェーズ）
- 子牛のスプライト生成（子牛はシンプルなスプライトで仮置きでよい）

---

## 完成の目安

以下が動作すればフェーズ1完了：

1. ホーム画面から牛舎に遷移できる
2. 背景画像の上に主人公と牛が表示されている
3. 牛房をタップすると主人公が歩いて近づく
4. 詳細シートが開いて牛の情報が表示される
5. シートを閉じると主人公が通路に戻る
6. 💩スペースと😷アイコンのスペースが確保されている（非表示）
