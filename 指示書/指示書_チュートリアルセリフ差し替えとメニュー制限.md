# 指示書：プロローグのチュートリアルセリフ差し替えと初期メニュー制限

## これは何の作業か

1. prologue.htmlのチュートリアル導入セリフを差し替える
2. 序盤（Day1〜2）は「餌を集めにいく」以外のメニューをグレーアウトする

---

## ①prologue.htmlのセリフ差し替え

以下の部分を差し替える：

```js
// === 暗転→タイトルロゴ ===
{
  isFadeToTitle: true,
},
```

この直後から`isLast: true`のシーンまでを、以下に丸ごと置き換える：

## ②序盤のメニュー制限（home.html）

### 対象メニューボタン

ホーム画面下部の4ボタンのうち、以下の3つをDay1〜2の間グレーアウトして押せないようにする：

| ボタン | Day1〜2 | Day3以降 |
|---|---|---|
| 餌を集めにいく | **有効**（常に押せる） | 有効 |
| 牛舎 | グレーアウト（押せない） | 有効 |
| ショップ | グレーアウト（押せない） | 有効 |
| 日誌 | グレーアウト（押せない） | 有効 |
| 呪文 | グレーアウト（押せない） | 有効 |

### 実装方法

home.htmlの描画時に`state.day`を参照して制御する。

```js
const isTutorial = state.day <= 2;
// isTutorialがtrueの間、牛舎・ショップ・日誌・呪文ボタンをdisabled状態にする
```

グレーアウト時はタップしても何も起きない。
エラーや説明テキストは不要（見た目がグレーになるだけでよい）。

4. Day3以降は全ボタンが有効になっている
