# 指示書：共通ヘッダーの実装

## これは何の作業か

全画面に統一された共通ヘッダーバーを実装する。
現在各画面がバラバラのヘッダーを持っているので、共通コンポーネントに統一する。

---

## ヘッダーのデザイン

header_black.htmlを参照すること。

```
背景色: rgba(10, 10, 10, 0.88)
下ボーダー: 1px solid #444
パディング: 6px 12px

左側:
  - 年月テキスト（例: "1年目　春　3月下旬"）
    font-size: 11px, color: #c8c080
  - Dayテキスト（例: "Day 6"）
    font-size: 10px, color: #708060

右側（横並び、gap: 10px）:
  - 🔮 [魔力残量]   color: #90d060, font-size: 12px bold
  - 💰 [ゴールド]G  color: #f0d060, font-size: 12px bold
  - [ラップ藁アイコン] [ラップ藁数]  color: #e0e0d0, font-size: 12px bold
```

ラップ藁アイコンは `assets/images/icon_wrap_wara.png` を使用。
width: 18px, height: 18px, image-rendering: pixelated

---

## 共通化の方法

`assets/js/common.js` に以下の関数を追加する：

```js
function renderHeader(targetElementId) {
  const state = loadState();
  const dateInfo = formatDate(state.day);
  const manaRemaining = calcTotalMagic(state.cows) - (state.manaUsed || 0);
  const el = document.getElementById(targetElementId);
  if (!el) return;
  el.innerHTML = `
    <div class="header-left">
      <div class="header-date">${dateInfo.text.replace('　', '<br>')}</div>
      <div class="header-day">Day ${state.day}</div>
    </div>
    <div class="header-stats">
      <span class="stat-mana">🔮 ${Math.max(0, manaRemaining)}</span>
      <span class="stat-gold">💰 ${state.money}G</span>
      <span class="stat-wara">
        <img src="assets/images/icon_wrap_wara.png" class="wrap-icon"> ${state.wrapWara || 0}
      </span>
    </div>
  `;
}
```

## 各画面への適用

以下の画面のヘッダーを共通ヘッダーに差し替える：

| 画面 | 対応 |
|---|---|
| home.html | 差し替え |
| barn.html | 差し替え |
| map.html | 差し替え |
| date_change.html | 差し替え |
| upkeep.html | 差し替え |
| feeding.html | 差し替え |
| mana_gain.html | 差し替え |
| shop.html（未実装） | 実装時から共通ヘッダーを使う |
| **explore.html** | **差し替えない**（神経衰弱画面はそのままにする） |

各画面のHTML側：
```html
<div id="gameHeader" class="game-header"></div>
```

各画面のJS側（ページ読み込み時）：
```js
renderHeader('gameHeader');
```
