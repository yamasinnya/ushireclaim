## 命名システム（共通モジュール、将来の子牛命名にも流用）

### バリデーションルール

```js
// assets/js/naming.js として新規作成

const NAMING_RULES = {
  female: {
    pattern: /^[ぁ-ん0-9０-９]+$/,  // ひらがな・数字のみ
    maxLength: 8,
    placeholder: 'ひらがな・数字、8文字まで',
    hint: '例：はるか、もも3'
  },
  male: {
    pattern: /^[一-龯0-9０-９]+$/,  // 漢字・数字のみ
    maxLength: 8,
    placeholder: '漢字・数字、8文字まで',
    hint: '例：黒山1、大雪'
  }
};

function validateCowName(name, gender) {
  const rule = NAMING_RULES[gender];
  if (!name || name.length === 0) return { valid: false, error: '名前を入力してください' };
  if (name.length > rule.maxLength) return { valid: false, error: `${rule.maxLength}文字以内にしてください` };
  if (!rule.pattern.test(name)) return { valid: false, error: gender === 'female' ? 'ひらがなと数字のみ使えます' : '漢字と数字のみ使えます' };
  return { valid: true };
}
```

プロローグでの最初の牛は **female（メス）** で固定。

## スプライトの状態定義

| キー | 見た目の意図 | 実装の目安 |
|---|---|---|
| `player: 'lying'` | 地面に横たわっている | playerスプライトを下部に配置、90度回転か縦に縮小で近似 |
| `player: 'sitting'` | 起き上がりかけ | playerスプライトを通常の70%サイズ、下寄りに配置 |
| `player: 'standing'` | 通常の立ち姿 | 通常配置 |
| `player: 'face_cow'` | 牛の方を向く | playerの左右反転（牛が左にいれば左向きに） |
| `cow: 'far'` | 遠くにいる | 通常の60%サイズ、画面右奥寄り |
| `cow: 'near'` | 近くにいる | 通常サイズ、プレイヤーの左横 |
| `cow: 'nuzzle'` | 鼻を近づけている | nearより少しプレイヤー寄り |
| `filter: 'dark'` | 夜明け前の暗さ | 画面全体に半透明の黒オーバーレイ（opacity: 0.5程度） |
| `filter: 'brightening'` | じわっと明るくなる | darkからnormalへのCSSトランジション（2秒程度） |
| `filter: 'normal'` | 通常の明るさ | オーバーレイなし |

## 名前入力UIの仕様

`isNaming: true` のシーンで、会話ボックスの代わりに名前入力UIを表示する。

```
┌─────────────────────────┐
│ 名前をつけてあげよう     │
│                         │
│ ┌───────────────────┐  │
│ │                   │  │  ← テキスト入力欄
│ └───────────────────┘  │
│ ひらがな・数字、8文字まで │  ← hint
│                         │
│    [決める]             │  ← 決定ボタン
└─────────────────────────┘
```

- 入力欄はスマホのソフトキーボードが出る形式（`<input type="text">` or `<input type="text" inputmode="hiragana">`）
- バリデーションはリアルタイムで実施。NGの場合は決定ボタンをグレーアウト
- 決定ボタンを押すと `validateCowName(name, 'female')` でチェック
- OK なら `state.cows[0].name = name` をセーブして次のシーンへ
- 名前は以降の `{cowName}` プレースホルダーで参照

---

## `{cowName}` の置換

`text` フィールドに `{cowName}` が含まれる場合、表示時に `state.cows[0].name` で置換する。
（例：「{cowName}、か。……いい名前だな」→「ふうか、か。……いい名前だな」）

---

## 命名モジュールの再利用性

`naming.js` の `validateCowName(name, gender)` は将来の子牛命名でも使い回す。
呼び出し側でgenderを渡せばルールが切り替わる設計にしておくこと。

---

## セリフの t() 管理

配列の `text` は全て t() 経由で取得する。
キー名は `prologue_00`〜`prologue_XX` の連番。

`{cowName}` は t() のキーのテキスト内に文字列として含めておき、表示時に置換処理をかける。

## 完成の目安

1. 既存のプロローグが新しいシナリオ本文に差し替わっている
2. スプライトがシーンごとに状態を変える（lying→sitting→standing）
3. 夜明けの明るさ変化が演出されている
4. 名前入力UIが表示され、ひらがな・数字のみ・8文字以内のバリデーションが動く
5. 入力した名前がセーブされ、「{cowName}、か」のセリフで正しく表示される
6. 暗転→タイトルロゴ→チュートリアル導入の流れが動く

## 追記：最初の牛の初期データ

名前入力が完了した時点で、セーブデータに以下の牛データをセットする。

```js
{
  id: 'cow_001',
  name: /* 入力された名前 */,
  gender: 'female',
  ageDays: 120,          // 5歳（1年=24日換算）
  seed: Math.floor(Math.random() * 99999) + 1,  // ブチ模様はランダム
  condition: 7,
  quality: 2,            // 可
  qualityPoint: 0,
  skill: 'sixth_sense',  // 第六感（実装済み）
  type: 'mother',
  pregnantDay: 5,        // 妊娠5日目スタート（出産予定日まで13日）
  poopCount: 0,
  diseaseAlert: false,
}
```

`pregnantDay` は「妊娠してから何日経過したか」。毎日アップキープで+1し、18日に達したら出産予定日判定に入る既存ロジックと接続すること。
