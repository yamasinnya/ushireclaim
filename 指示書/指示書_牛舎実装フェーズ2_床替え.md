# 指示書：牛舎画面の実装（フェーズ2）床替えシステム

## これは何の作業か

フェーズ1で確保した💩スペースを実際に機能させる。
💩が溜まって、タップすると床替えができる、までを実装する。

---

## 💩の増加ロジック

アップキープ処理（date_change.html または upkeep.html）に以下を追加：

- 毎日アップキープ時に全母牛の `poop_count` を +1 する
- 上限は 4（4個で止まる）
- 子牛は対象外（母牛のみ）

---

## 💩の表示

各牛房の右下エリアに `poop_count` の数だけ💩を表示する。
（フェーズ1で確保したスペースをそのまま使う）

- 0個：何も表示しない
- 1〜4個：💩を横並びで表示

---

## 💩タップで床替え

💩アイコンをタップすると床替えの選択肢シートを開く。

### 選択肢シートの内容

```
🧹 床替えしますか？

[①自分でやる]     魔力 4 消費
[②ショップに頼む]  1000G（1年目は500G）
[キャンセル]
```

- 魔力が4未満の場合、①はグレーアウトして選択不可
- 所持金が足りない場合、②はグレーアウトして選択不可
- どちらを選んでも実行後に `poop_count` を 0 にリセット

### 3個・4個の場合の追加表示

- 3個以上：選択肢シート上部に牛のコメントを表示
  - `t("poop_comment_3")` → 「なんか臭うな…そろそろ掃除してくれ」
- 4個：品質ポイントを毎日 -1 する処理も追加（アップキープ時）

---

## セーブデータへの反映

アップキープ時：
```js
state.cows.forEach(cow => {
  if (cow.type === 'mother') {
    cow.poop_count = Math.min(4, cow.poop_count + 1);
  }
  if (cow.poop_count >= 4) {
    cow.qualityPoint = Math.max(0, cow.qualityPoint - 1);
  }
});
```

床替え実行時：
```js
cow.poop_count = 0;
```
