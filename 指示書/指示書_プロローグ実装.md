# 指示書：プロローグ画面の実装（prologue.html）

## これは何の作業か

タイトル画面の「はじめから」を押した後に表示されるプロローグシーンを実装する。
**新しい画像は使わない。** 既存スプライトと背景画像をそのまま流用する。

---

## 画面の構成

既存のホーム画面背景画像を使い、その上にスプライトを重ねる。

```
[背景：ホーム画面と同じ画像、全体的に暗くフィルターをかける]
  ↓ 重ねる
[牛スプライト（既存）]
[主人公スプライト（既存）]
[軽トラスプライト（既存）]
  ↓ 重ねる
[下部：会話ボックス（名前＋セリフ）]
[画面全体：タップで次へ]
```

---

---

## スプライトの動き

複雑なアニメーションは不要。シーンが変わるたびに位置・表示状態を切り替えるだけでよい。

| 状態 | 実装方法 |
|---|---|
| 牛 far | 画面右奥に小さく表示 |
| 牛 near | 主人公の隣に表示 |
| 牛 nuzzle | nearより少し主人公に近い位置 |
| 主人公 lying | 地面付近に横向きで表示（回転かオフセットで近似） |
| 主人公 sitting | 地面付近に通常サイズで表示 |
| 主人公 standing | 通常の立ち位置 |
| filter dark | `backdrop-filter` or オーバーレイで暗め |
| filter normal | 通常（フィルターなし） |

---

## 会話ボックスのデザイン

画面下部に固定。他の画面の詳細シートと統一したデザイン。

```
┌─────────────────────────┐
│ 軽トラ                   │  ← 話者名（nullの場合はイタリック表示）
│ まあ、細けぇ話は追い追いだ │  ← セリフ本文
│               [タップで次へ]│
└─────────────────────────┘
```

- ト書き（isNarration: true）は話者名なし、テキストをイタリック・中央寄せ
- 選択肢（isChoice: true）は「……（うなずく）」ボタンを中央に表示
- タイトルロゴ表示（isFadeToTitle: true）は暗転→タイトル画像をフェードイン→2秒後に自動で次へ
- 最後のセリフの後（isLast: true）は home.html へ遷移

---

## セリフは t() 経由で取得

配列の `text` と `choiceText` はja.jsonのキーとして管理する。
キー名は `prologue_00`〜`prologue_XX` の連番でよい。

---

## 共通ヘッダーについて

プロローグ中はヘッダーを表示しない（全画面でシーンに集中させる）。

---

## 遷移フロー

```
title.html「はじめから」
　↓
prologue.html（このファイル）
　↓（最後のセリフをタップ）
home.html
```

---

## 触らないこと

- home.html・barn.html・その他既存画面
- セリフの内容（上記の通り実装。変更する場合はやましさんに確認すること）

---

## 完成の目安

1. タイトルの「はじめから」でプロローグが始まる
2. タップで順番にセリフが進む
3. ト書きとセリフが見た目で区別できる
4. 選択肢「……（うなずく）」ボタンが表示される
5. 暗転→タイトルロゴ表示の演出がある
6. 最後のセリフでホーム画面に遷移する

---

## 確認してほしいこと（実装後に報告）

- スプライトの位置調整（lying/sitting/standingの見た目がどうなったか）
- タイトルロゴの表示方法（画像ファイル名を確認して使用すること）
- 追加した t() キー名の一覧
- 不明点は実装を止めてやましさんに確認すること
