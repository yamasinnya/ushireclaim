# 指示書：ショップ（街）画面の実装（枠のみ）

## これは何の作業か

街の背景画像を使って、5軒の店舗に入れる画面を作る。
各店舗の具体的な機能はまだ実装しない。「枠と導線」だけを作るフェーズ。

---

## 画面遷移

```
home.html の「ショップ」ボタン
　↓
street.html（街の背景画像）
　↓ 各建物をタップ
各店舗のシート（下からスライドアップ）
　↓ 閉じるボタン
street.html に戻る
```

---

## street.html の構成

### 背景画像
- GPTが生成した街の背景画像（1000004222.png）を使用
- 牛舎と同じアプローチ：背景画像の上にタップ判定レイヤーを重ねる
- 画像は縦長・スマホ幅に合わせてリサイズ

### 共通ヘッダー
- 他の画面と同じ共通ヘッダー（renderHeader）を使用

### タップ判定
各建物の座標はClaude Codeが画像を見て判断して設定すること。
建物名と対応する店舗IDは以下：

| 建物 | 店舗ID |
|---|---|
| 何でも屋 | shop_nandemo |
| 獣医 | shop_juui |
| コントラクター | shop_contractor |
| 建設屋 | shop_kensetsu |
| 競り市場 | shop_seri |

### 戻るボタン
- 「← 牧場へ戻る」ボタンで home.html へ遷移

---

## 各店舗のシート

建物タップで下からスライドアップするシート。
牛舎の詳細シートと同じデザイン・アニメーションを流用。

### シートの共通構成

```
[店舗名]
[一言説明]
──────────────
[商品①] [準備中ボタン]
[商品②] [準備中ボタン]
...
──────────────
[閉じる]
```

### 各店舗の内容

**何でも屋**
- 説明：「農作業の代行をします。」
- 商品：床替え代行（準備中）

**獣医**
- 説明：「牛の健康をサポートします。」
- 商品：種付け 高級（準備中）
- 商品：種付け 普通（準備中）
- 商品：種付け 安い（準備中）
- 商品：病気治療（準備中）

**コントラクター**
- 説明：「農業機械を使った作業を請け負います。」
- 商品：ラップ藁 6日分（準備中）
- 商品：ラップ藁 12日分（準備中）
- 商品：ラップ藁 24日分（準備中）

**建設屋**
- 説明：「施設の建設・IT化をお手伝いします。」
- 商品：魔法の堆肥舎（準備中）
- 商品：牛舎拡張（準備中）
- 商品：IT化設備カタログ（準備中）※「行商人が来た時に購入できます」と注記

**競り市場**
- 説明：「子牛の競りへの参加と、母牛の購入ができます。」
- 商品：競り出品申込（準備中）
- 商品：母牛を購入する（準備中）

### 準備中ボタンのデザイン

```css
.btn-coming-soon {
  background: #2a2a2a;
  border: 1px solid #444;
  color: #555;
  padding: 8px 16px;
  border-radius: 4px;
```

タップしても何も起きない（disabled状態）。

## ファイル構成

```
street.html        ← 今回新規作成
assets/js/street.js ← 街のロジック（タップ判定・シート表示）
```

common.js・common.css は既存のものを使用。

---

## デザイン指針

- シートのデザインは牛舎の詳細シート（木目・暗め）と統一
- 「準備中」は暗くグレーアウトして、実装済みと明確に区別
- **文字列はすべて t() 経由**（ja.jsonに追記）

---

## 触らないこと

- 各店舗の実際の購入ロジック（全て別フェーズ）
- home.htmlの「ショップ」ボタンのリンク先は `street.html` に変更するだけでよい

---

## 完成の目安

1. ホーム画面の「ショップ」ボタンで街画面に遷移する
2. 街の背景画像が表示される
3. 各建物をタップするとシートが開く
4. シートに店舗名・説明・準備中の商品一覧が表示される
5. 閉じるボタンでシートが閉じる

---

## 確認してほしいこと（実装後に報告）

- 追加した t() キー名の一覧
- 不明点は実装を止めてやましさんに確認すること
