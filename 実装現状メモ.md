## メインセーブのデータ構造（実装ベース）

`localStorage['yamayama_loop_v1']`、`assets/js/common.js`の`LOOP_DEFAULT_STATE`が実体。

```js
{
  version: 1,
  day: 1,
  money: 0,
  wrapWara: 0,
  manaUsed: 0,
  grassStock: 0,
  qualityPoint: 0,   // ★ state直下。牛ごとではない（後述）
  buildings: {
    gyusha_small: true,
  },
  cows: [
    {
      id: 'cow_001',
      name: 'ふうか',
      gender: 'female',
      age: 48,          // ★ フィールド名は age（ageDaysではない）
      seed: 1234,
      condition: 6,
      quality: 2,
      skill: 'zenno',   // ★ 識別子は herdboys_eye / trace / roku / zenno の4種のみ
      type: 'mother',
      pregnantDay: 0,   // ★ 増加するだけで出産判定ロジックは未実装
      poopCount: 0,
      diseaseAlert: false,
    },
  ],
}
```
